🚀 HOW TO USE:

1. Download the entire "amnen" folder from this chat

2. Extract to your computer

3. Copy the "amnen" folder to:
   Windows: C:\xampp\htdocs\
   Mac: /Applications/XAMPP/htdocs/
   Linux: /opt/xampp/htdocs/

4. Start XAMPP:
   - Open XAMPP Control Panel
   - Start Apache
   - Start MySQL

5. Initialize Database:
   - Go to: http://localhost/amnen/setup.php
   - Wait for green checkmarks

6. Login:
   - Go to: http://localhost/amnen/index.php
   - Username: admin
   - Password: Admin@123

7. You're done! Start using the system.
