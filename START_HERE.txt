🚀 INSTALLATION STEPS
═══════════════════════════════════════════════════════════════════════════════

STEP 1: EXTRACT TO XAMPP
─────────────────────────
Windows:
  1. Open File Explorer
  2. Go to: C:\xampp\htdocs\
  3. Paste the "amnen" folder here

Result: C:\xampp\htdocs\amnen\ (with all files inside)

STEP 2: START XAMPP SERVICES
───────────────────────────
1. Open XAMPP Control Panel
2. Click "Start" next to Apache
   Wait for: (Running) ✓
3. Click "Start" next to MySQL
   Wait for: (Running) ✓

Both should show green checkmarks.

STEP 3: INITIALIZE DATABASE
──────────────────────────
1. Open web browser
2. Go to: http://localhost/amnen/setup.php
3. Wait for the page to load
4. You should see green ✓ checkmarks:
   ✓ Loaded .env file
   ✓ Config loaded
   ✓ Database connection established
   ✓ Database tables created successfully
   ✓ All classes loaded
   ✓ Admin user created

If all green → Setup successful!

STEP 4: LOGIN TO SYSTEM
──────────────────────
1. Go to: http://localhost/amnen/index.php
2. Click "Login"
3. Enter:
   Username: admin
   Password: Admin@123
4. Click "Login"

You're now in the Admin Dashboard! 🎉

📞 TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

Problem: "Cannot connect to database"
Solution:
  1. Check MySQL is running in XAMPP (green ✓)
  2. Run http://localhost/amnen/setup.php again
  3. Check .env file exists and is readable

Problem: "Tables don't exist"
Solution:
  1. Go to http://localhost/amnen/setup.php
  2. Let it recreate all tables
  3. Refresh page to confirm

Problem: "Login not working"
Solution:
  1. Run http://localhost/amnen/diagnostic.php
  2. Check if admin user exists in database
  3. Run setup.php to recreate admin
  4. Clear browser cookies

Problem: "404 Page not found"
Solution:
  1. Check URL is exactly: http://localhost/amnen/index.php
  2. Check folder is at: C:\xampp\htdocs\amnen\
  3. Ensure Apache is running

Problem: "Blank white page"
Solution:
  1. Check diagnostic: http://localhost/amnen/diagnostic.php
  2. Look for PHP error messages
  3. Check .env file exists
  4. Run setup.php again
