🚀 READY TO INSTALL ON XAMPP:

1. EXTRACT FILES
   → Copy entire 'amnen' folder to C:\xampp\htdocs\
   → Result: C:\xampp\htdocs\amnen\ (all files here)

2. START SERVICES
   → Open XAMPP Control Panel
   → Click Start next to Apache
   → Click Start next to MySQL
   → Wait for green checkmarks

3. RUN SETUP
   → Open browser: http://localhost/amnen/setup.php
   → Wait for all ✓ green checkmarks
   → System will auto-create database & tables

4. LOGIN & USE
   → Go to: http://localhost/amnen/index.php
   → Username: admin
   → Password: Admin@123
   → You're in!

❓ TROUBLESHOOTING:

"Database connection failed"
  → Check if MySQL is running in XAMPP
  → Run setup.php again
  → Check .env file

"Can't see tables"
  → Go to http://localhost/amnen/setup.php
  → Let it recreate all tables

"Login doesn't work"
  → Run diagnostic.php to check status
  → Run setup.php to recreate admin
  → Clear browser cookies

"Page not found"
  → URL must be: http://localhost/amnen/index.php
  → Folder must be: C:\xampp\htdocs\amnen\
  → Check that files were extracted correctly
